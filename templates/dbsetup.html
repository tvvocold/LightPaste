MySQL
General:
  • Make sure your installation of PHP supports MySQL functionality
config.ini:
  • Set DATABASE_DSN to "mysql:host=localhost;port=3306;dbname=lightpaste" and replace each value where necessary
  • Set DATABASE_USER to the username that you want Light Paste to use when accessing the database
  • Set DATABASE_PASSWORD to the password used by DATABASE_USER
SQLite
General:
  • Make sure your installation of PHP supports SQLite functionality
config.ini:
  • Set DATABASE_DSN to a location that your installation of PHP has read and right access to. For example: "sqlite:/path/to/database.sqlite"
