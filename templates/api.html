Creating a Paste
To create a new paste, send a POST request to http://lightpaste.com/api/paste. Light Paste will check the request for the following data:
KeyValueRequired
textContents of the pasteYes
languagePaste languageNo
visibilityVisibility of the pasteNo
expirationWhen the paste should expireNo
If the paste was successfully created, Light Paste send a response in the following format:
HTTP/1.1 201 Created
Content-Type: application/json; charset=utf-8

{
    "paste_id", "ZxMo14a2"
}
Retrieving a Paste
To retrieve a paste, send a GET request to http://lightpaste.com/api/paste/*paste_id* and replace *paste_id* with the id of the paste that you are looking for. If the request was successful, Light Paste will send a response in the following format:
HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

{
    "id", "ZxMo14a2"
    "text", "**encoded text**"
    "language", "lua"
    "time", "1393114519"
    "views", "10"
    "md5", "96a6eddfb4173bcad25228990720a2d9"
    "sha1", "63c4e4e27cf625a637abe04f0a9faa9f08a2e310"
}
Note: The text field listed in the response body will contain the paste's text encoded in Base 64.
Errors
When an API request is unsuccessful or encounters an error, Light Paste will send a response in following format:
HTTP/1.1 **status code**
Content-Type: application/json; charset=utf-8

{
    "error", "true"
    "error_message", "**error_message**"
}
